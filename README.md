## SCRIPT DE PUNTOS DE RESTAURACIÓN

Script en PowerShell para gestionar puntos de restauración del sistema en Windows.

OPCIONES:
1. Inicio - Muestra las funciones del script
2. Crear punto de restauración - Crea un nuevo punto con nombre personalizado
3. Comprobar todos los puntos - Lista todos los puntos existentes
4. Comprobar punto específico - Busca un punto por su nombre
5. Salir - Cierra el script

REQUISITOS:
- Windows 10/11
- PowerShell 5.0 o superior
- Permisos de administrador
